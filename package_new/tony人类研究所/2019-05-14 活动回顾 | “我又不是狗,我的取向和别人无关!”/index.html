/ONE/


到观影会的时候影片已经到了主角被送入同性恋转换治疗场所，后续的剧情展开围绕着主角与牧师、主角与父亲的互动进行。

对于杰拉德而言，他在机构里面曾经尝试过将自己同性恋的本性消解，他想融入同是他认识的病人群体里面再正常化。事实上，他所见的是牧师要求卡梅伦承认自己是同性恋是恶心的行径，机构的员工鄙夷式地谩骂“基佬”，致使卡梅伦最后自杀的是牧师借助个人的愤怒、上帝的能力迫使个人承认自己的天性是病的规训。整个同性恋转换治疗场所类似一个小型监狱对犯人的规训，借助正统合理性的基督教义宣达破坏了以自己为中心的差序关系的行为是错的，这种规训的逻辑推演以及道德范畴归结到了上帝的正义，但是同性恋是天性的逻辑却不被倡导表达。杰拉德无疑是失望的，因此最终爆发与牧师的冲突，并让母亲帮助自己逃离这里。杰拉德的幸运，却是多少逃不离的人长时间“play the part”的遭受心灵煎熬的不幸。

杰拉德和父亲的对话，每一次都让我感觉像是在爆发的边缘点，但是父亲在事态发展得超乎意料的过程中，逐渐认识到自己解决问题的方式太偏激。他等到杰拉德陈述“我是同性恋无法改变，我是你儿子无法改变”，他内心才意识到放弃和接受是仅有两种最合适的选择，即使放弃父子关系坚持自己的信仰也是选择之一，也会比强制扭转治疗来得更尊重儿子。最后儿子与父亲以笔的传授达成说服，父亲作出尝试理解的决定为影片画上句号。

按照霍尔对电视话语编码与解码的观点，信息是以符号为载体形式的传播，类似奇葩说，冲突和说服是影片的两个建构的符号。影片展现了霍尔三个假想立场之二：抵制式-对抗式的立场以及协商式立场。牧师与杰拉德的对话导演运用了对抗式的立场，牧师针对同性恋的少数情形对在场病人的打压，以罪和负面感官给同性恋标签化，进而引发杰拉德与牧师关于愤怒逻辑归谬的冲突。父亲与杰拉德的对话则展现了协商式的立场，父亲的牧师身份与儿子的同性恋情形在有限包容度的家庭氛围中，缓慢展开对垒，父亲从避而不谈到与儿子共同直面事件，他慢慢达成自己信仰的包容。

《被抹去的男孩》中“被抹去” 能指说的主角在同性恋治疗机构里面隔离开外界大众的经历；所指说的是是男孩的同性恋身份被大众不认同因而说是被抹去，电影中被抹去的情景没有出现，结局也有光明的前景，但这也意味变数仍在以及治疗机构里的病人阴霾未散，现实还是在盖在纸背面的记忆。

“美国至今依然有36个州合法允许改造学校存在，而且每年有70万人在遭受改造”



在21世纪，我们还是能看到一些歧视同性恋的行为。


不过，也不外乎被多看几眼议论几句，极端点的可能会被“竖中指”或吐个口水。


但是这些遭遇，相比上个世纪同性恋面临的处境根本就是小巫见大巫。


现在我们都知道，同性恋不是疾病，不能改变也无需治疗。


但在上个世纪几乎所有人都坚信同性恋就是病，为了“治愈”同性恋，各种残忍的治疗手段更是层出不穷。


“磁爆步兵”杨永信的“十万伏特”都只是初阶玩法。


此外，还有“睾丸移植”、“直接切除脑前额叶”等骇人听闻的同性恋“矫正治疗”。





在这荒诞的“同性恋治疗”背后，更是一段让人毛骨悚然的医学黑历史。


而这段黑历史中，“人工智能之父”艾伦·图灵就是最有名的受害者，这也是“赦免同性恋法案”为什么叫《图灵法案》的原因。

 


0

二战期间，他曾帮英国破译了纳粹密码，在诺曼底登陆等军事行动中发挥了重要作用。


虽然他的一生功勋显赫，但还是因为同性恋的身份被强制接受激素治疗。


当时他被注射的是一种叫已烯雌酚的激素类药物，是人工合成的雌性激素。


但与其说图灵接受的是“激素治疗”，倒不如直接说是“化学阉割”，因为它与现代的化学阉割根本没什么两样。


这些激素药不但让他在生理上无法勃起，还使其胸部开始像女性一样开始发育。


在巨大的压力下，图灵开始陷入重度抑郁，最终用毒苹果结束了自己性命。





关于“同性恋治疗”的伪科学随着时代与科学的发展不攻自破，人们也慢慢意识到这种不人道的治疗给同性恋人群带来的不仅不是“健康”，反而是一种巨大的伤害。




1990年，世界卫生组织（WHO）正式将同性恋从疾病名册删去

2001年，中国将同性恋去病化。

2016年末，英国政府决定启动《图灵法案（Alan Turing law）》，赦免曾因犯“严重猥亵法”而被定罪的6.5万名同性恋或双性恋男子。

目前，全球已有超过20个国家规定同性恋婚姻合法，同性恋运动在这个世纪算是取得了巨大成功。

  


 但是这并不代表反同性恋的国家和团体就不复存在了。


  




2013年，鼓吹自己是最民主国家”的印度也重新决定同性恋为非法行为




2019年3月，东南亚国家文莱宣布《伊斯兰刑法》生效，同性恋性行为将被以石刑处死。这意味着文莱将成为亚洲首个将同性恋定为死罪的国家。




2019年5月，在巨大的国际压力下，文莱政府宣布将撤回原先的决定，暂停死刑的执行，并强调文莱是一个公平国家，观光环境安全且友好。
