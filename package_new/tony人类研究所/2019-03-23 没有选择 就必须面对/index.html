没有选择 就必须面对

原创 Sam tony人类研究所
2019年03月23日 22:24


“毒品 种族 同性恋 暴力 黑人社会”









男孩“希隆”的母亲是瘾君子，命运中冥冥注定出现一个启蒙人，却正是那个贩毒给母亲的男人胡安。

胡安把他带回家和妻子特蕾莎细心照顾他，陪他聊天给了他家的感觉。从小失去父亲的希隆，在母亲那里又长期得不到关爱，内心无依无靠的不安，在遇到胡安之后终于有了避风的港湾。






青春期的男孩总是对性充满了好奇与探索，生性懦弱的希隆身边没有女生，唯一的朋友就是凯文。

两人之间的兄弟情却渐渐变的暧昧，一天在班里恶霸的威逼下，凯文为了掩饰内心的同性恋取向出手揍了希隆。




长大后希隆再次被欺负这次他依旧没有还手，也不恨凯文，希隆孤僻的性格以及时常备受欺凌的境遇下。

唯一的朋友凯文对于他可以说弥足珍贵，尤其是胡安去世之后，凯文更是他唯一可以谈心和得到宽慰的人，心里唯一在乎的人。

所以后来在海滩边的吐露爱意和亲密接触也就水到渠成。



凯文本来是敢于抗争的人，但在成长中渐渐被生活磨去了棱角。

中学时在同学们的胁迫以及社会对同性恋歧视的压力下，最终背叛殴打了希隆，这给了希隆戳心的一击，使得希隆在沉默中爆发，用强大的外表武装保护自己，不再依靠其他人。

他开始健身变强壮，成为了第二个胡安。





很多年之后凯文向胡安的妻子特雷莎要了希隆的电话，并打电话告诉希隆他很想他。

其实希隆也很想念凯文，时常梦见凯文。



闪烁的眼神与微笑，欲言又止的微妙暧昧，介于知己与恋人之间，无论同性异性，纪念青春期因极度孤独、一度亲密而狂喜的瞬间。





希隆没有父亲，母亲吸毒，为了买毒品去卖淫；被周围的孩子欺凌；童年给他带来父爱般关怀的胡安却是毒枭，是母亲吸毒的罪魁祸首；青年时唯一的好友与至爱背叛、伤害了他。








我们虽然极少有这些极端的经历，然而类似的伤痛也比比皆是。父母的责难、父母的争吵、做错事时的打骂、青春期时无疾而终的友情、毕业后失去联系的好友。







当然《月光下的男孩》并不是消极的陈述这些苦楚，希隆在学校终于决定了自己要成为一个怎样的人，奋起反抗还击了对他施暴的人。


一直到十多年后他用强大的外表武装起自己免受小时候的欺凌，成为了自己心目中的胡安一样的男人。





没有选择，就必须面对。这部电影讲述的不是鲜血四溅的残酷，而是近在咫尺却相对无言的绝望，以及转身离开后重整旗鼓的勇气。
