给恐同老师寄封信



我们的志愿者无意中发现在学校中有几位老师对LGBT存在偏见，同时在课堂上发表了不恰当的言论，于是几位志愿者执笔了这封信。

亲爱的陈老师：

您好！

十分感谢您在百忙中抽出时间阅读这封信件，作为您的学生，同时也作为性少数群体中的一员，我有话想对您说。

在一次思想道德修养课上，您谈到同性恋话题。面对整个专业一百多名同学，您说到：“中国的同性恋比大熊猫还稀少”“同性恋是心理疾病”。您的言论让我震惊；从未想到如此不经推敲的言论会在大学课堂中出现。这件事也让我了解到，校园成为恐同欺凌的多发地是事出有因的。

据2004年卫生部发布的《同性恋白皮书》显示，仅处于性活跃期的男同性恋的数量就超过1000万。早在1990年，世界卫生组织（WHO）正式将同性恋从疾病名册中去除。2001年，在中华精神科学会推出的第三版中国精神疾病诊断标准（CCMD-3）将同性恋从精神疾病分类中删除。在调查和数据面前，曲解同性恋的言论不攻自破。但这种言论产生的可怕后果却无法挽回。

2006年，广西某初中生被教师曝光性向当众羞辱，使他离家出走；2007年，成都一名跨性别（女跨男）初中生在大冬天被迫当众洗屁股；2016年3月安徽黄山田家炳中学一名女同性恋学生被下春药。这些校园恐同欺凌不仅反映了相关教育的缺失和校园环境的不友善，同时还敲响了警惕校园恐同的警钟。这也是我们写这封信给您的原因——我们希望老师们在课堂上提及相关话题时能更加理智客观，考虑性少数学生的感受，在充分了解客观事实的前提下对学生做出正确指引。

虽然恐同的观点我并不赞同，但我尊重老师在这件事情上拥有不同看法的权利。可是，鼓吹仇恨与歧视并不符合大学开放包容的精神气质。三尺讲台之上，您是课堂上的权威；为人师表，您是学子们的表率。作为德高望重的大学老师，您的一言一行都会对讲台下的学生产生深远影响。有时，几句不甚恰当的言论就有可能对处于自我认同困惑期的学生施加压力，这种来自课堂的压力与来自家庭、社会、伦理道德以及法律等多方面的压力共同作用，使得他们的生活、学习、人际交往等受到极大的影响，不少人因此患上不同程度的抑郁症，甚至出现自杀倾向。另外，这些言论还有可能歪曲学生对于性少数群体的认识，让不了解性少数群体的同学走上恐同恐跨的道路。据相关调查，在中国，有33%的学生都曾遭遇过基于性倾向和性别身份的校园欺凌。从某种程度而言，部分老师的不当言论助长了恐同欺凌的发生。校园里还有部分老师，本就研究社会科学，理应及时更新知识，跟上学术前沿，却依旧向同学们灌输“同性恋是病”这样错误的、过时的概念，甚至将这些错误理论编入教材。在自己的领域中犯这样低级的错误，实在令人惋惜。对社会而言，大学老师是重要的学术代表，如果连老师们都不能在这件问题上坚持客观公正的态度，为公众提供正确的价值导向，那么不同性向群体间的误解和歧视会日益加深，社会公正也更加难以得到实现。

宪法赋予每个公民言论的自由，但言论自由并不是没有边际的自由。恐同言论涉及侮辱、恐吓、人身攻击，是对性少数群体实实在在的伤害，这也是为什么“仇恨言论”在很多国家遭到（法律或道德上）禁止和反对。演员吕丽萍在微博上发表的恐同言论引起了轩然大波。著名性学家李银河对于此事发声称，吕丽萍恐同言论愚昧落后，乃至反动，作为公众人物不可原谅。台湾电影金马奖组委会决定暂缓其颁奖。央视批评其言论，并称名人应该反思自己的公开言论对社会造成的影响。与名人一样，大学老师享有一定的社会影响力和公众话语权。所以，我建议老师在相关议题上谨言慎行，注重自己的公众形象。

另外，保障教育公平是老师的责任。老师有责任保障教育的公平公正，性别是一个重要的方面。《科尔曼报告》中提到，教育公平不仅是平等的受教育机会，还应该帮助学生克服其背景原因带来的学习障碍。就教育公平而言，性少数学生需要不仅是平等的教育资源，还需要一个不因其性取向而受到干扰的校园环境。在此，我向老师发出倡议：

1、 了解多元性/别知识，及时更新知识体系。

2、 考虑课堂上性少数同学的感受，友善对待您的性少数学生。

感谢老师您能耐心看完这封信，祝愿老师工作顺利，生活愉快。

顺颂

教安

您的学生
