What the world has done



1.

网络节目审核通则歧视同性恋案一审判决下达

2月23日，原告诉被告国家广播电视总局《网络视听节目内容审核通则》涉歧视同性恋一案于北京市第一中级人民法院开庭审理。10月23日，一审行政判决书下达，法院以事实与理由缺乏依据不予支持为由，驳回原告全部诉讼请求。原告已准备上诉。


2.

社交媒体清查同性恋内容遭网友抵制

4月13日，新浪微博管理员发布公告，将开展为期3个月的违规内容清查。因该清查行为误解、丑化与歧视同性恋，掀起大批网友批评与抵制。4月16日，微博宣布清查不再针对同性恋



3.

中国首次在联合国人权理事会普遍定期审议上正面回应LGBT+问题


11月6日，在日内瓦召开了中国普遍定期审议第三轮审议。中华人民共和国代表团首次正面回答了有关于中华人民共和国LGBT+权益的相关问题，表示“尊重LGBT+群体的健康权，并给予其平等的社会保障”、“保护进行性别重置手术的权利”与“现阶段政策是由我国的历史文化价值观决定”等发言。


4.

世界同性恋运动会在巴黎开幕

8月4日，第10届世界同性恋运动会在法国巴黎开幕，来自90个国家和地区的10317名运动员将参加本届世同会。法国体育部长Laura Flessel、巴黎市长Anne Hidalgo以及法国著名的时装设计师Jean-Paul Gaultier等出席开幕式。本届也是中国人出席人数最多的世同会。


5.

菲律宾总统支持同性伴侣立法


7月2日，尽管菲律宾民调显示61%的民众反对同性婚姻合法，总统杜特尔特发言人表示总统支持立法承认同性民事伴侣



6.

泰国政府内阁通过《同性伴侣法》草案

12月25日，泰国政府内阁批准《同性伴侣法》草案，并送交国会。若国会审议通过，将交泰国王室批准并公告后生效。泰国有望成为亚洲首个法律承认同性伴侣并以法律进行保护的国家。


7.

印度最高法院判决同性性行为非罪化

9月6日，印度最高法院推翻《印度刑法典》第377条对同性性行为定罪条款。印度同性性行为不再违法。

最后 和大家分享一首诗
来自伊丽莎白·毕肖普
写给她同性伴侣的一首情诗


失眠

月亮从妆台镜子中

望出一百万英里

（或许也带着骄傲，望着自己

但她从未，从未露出微笑）

至远远超越睡眠的地方，或者

她大概是个白昼睡眠者。

被宇宙抛弃了，

她会叫宇宙去见鬼，

她会找到一湾水，

或一面镜子，在上面居住。

所以把烦恼裹进蛛网吧

抛入水井深处

进入那个倒立的世界

那里，左边永远是右边，

影子其实是实体，

那里我们整夜醒着，

那里，天国清浅就如

此刻海洋深邃，并且你爱我。

同志仍须努力

写在最后

2018

感恩过去

感恩自己做过的选择

感恩自己走过的路

2019

即使前路再艰难

也要一直一起走下去
