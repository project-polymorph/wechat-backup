你根本不必要伟大

原创 荆言 tony人类研究所
2017年05月14日 23:29

又是一年母亲节，于是又能看见铺天盖地的“伟大”“为母则刚”“为母亲买买买”“爱”“默默付出”等等一系列喜闻乐见的非……阿不，是“超人”的词汇。


每每看见这些东西，我总是具有“举一反三”的思维去想：欸，我爸爸去哪了？为什么我的母亲不能是个普通的人？为什么为人母需要做出那么多牺牲却要被当作理所当然，最后只能换回一声“妈妈我爱你”这样就够了。这样就够了吗？


我们对于母亲节的认识远比父亲节多。当我们说起父亲，想到了“父爱如山”，想到了“背影”，想到了小时候的“骑高高”，这几乎是绝大部分父亲的词汇了。或许还有“家庭责任”，但是在现代词汇中，父亲在“家庭责任”中担当的是什么呢？除了双方都需要承担的家用，就只有体力活了吧？


但是母亲呢？


我们知道“怀胎十月”与生产时的歇斯底里，我们知道家务与“妈妈的味道”，我们还有更多她与我们在一起、她对我的关怀的细节。与父亲对比起来，大概就是：一位是公园的丰碑、一位是身旁的真人；一方是文字描述而另一方则字字泣血。


为什么同为一个“人”，母亲要承担多那么多？


我还知道的是，当成为一位母亲，她把自己分成了两份，每个人都把她与她的孩子捆绑在一起，当男人依旧可以放手发展自己的事业；


当成为一位母亲，她需要长长的休假来“生一场大病”，以至于她曾经的公司会选择下调甚至放弃她。


当一位女性成为母亲，每一代的她的身体里割下了一团肉与精华，而男人只是在睡梦中被上帝取下了一条肋骨，一劳永逸。而家人国家却连无痛分娩都不愿意为其推广，因为孕妇死亡率的降低。


当一位女性成为母亲，她被赞颂受苦，她对于家庭的付出理所当然，她不再是一位独立的人而不停地与其孩子捆绑，而她的收获，则是正常人都会做出的感恩与被夸大的伟大的帽子。


而事实上我们都知道，她不过是普通不过的，一个正常的人，而非被困家中的超人。


一个母亲，她并非得是那个怀抱着孩子的慈母，为什么她不能也是如父亲一样轻松地享受自己的生活呢？


嘿，我知道母亲非常辛苦，所以为什么不在正常的节日里为她分担呢。


环卫工人辛苦我们还知道少扔垃圾，那么妈妈365天的辛苦，家庭成员是否应该承担回自己错过的责任？


没有人是不辛苦的，但是一位女性一旦成为了母亲，她就承担了家庭与事业双倍的辛苦，而家庭这一部分，明明应该由父母双方再加上有了自理能力的孩子一起负担的，你在位了吗？


事实上，你并不是只有在这一天才想妈妈才爱妈妈，你的话语宣泄了，但需要在生活中行动；


而一直宣称“男人至死是少年”，同样的你并非只有在母亲的怀里才能做孩子，只要你保持内心的纯真与坚强，你可以每一刻都是个好孩子；


礼物或许需要一个借口去送，但是帮助母亲减少日常压力是每一天都应该做的事；


家务事辛苦了，让爸爸和我一起来吧我们轮流做，炒菜太烦，那也一人一次吧，或许还能增添些小情调；


别管我了，难道我向您的撒娇还不够吗，多想想你自己，如果你开心漂亮自信幸福，那才是我最骄傲的事情；


以后的家，还有谁缺席了呢？


以后的母亲节，我们让母亲不必成为一个“伟大”的母亲吧？
